Welcome to Equanimous Meow

This repo contains utilities for use with Slack.

slackarchive.py

Performs archival exports of Slack channels, as well as parsing exported history files to stitch together a continuous narrative labeled with user names in addition to ID strings.

slack_mefi.css

Slack doesn't support themes, but with the Stylish browser extension, this CSS file will theme the entire Slack web interface in a blue/green motif inspired by www.metafilter.com. Additional dialogs/pages are being added to the theme on an ongoing basis.
